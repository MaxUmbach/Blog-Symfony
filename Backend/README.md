# Symfony Blogging Platform

This is a **work in progress blogging web application** built with **Symfony**.  
The goal of this project is to create a basic blog system with user accounts, posts and database storage.

The web application itself runs locally (Symfony), while the **database is provided via Docker**.

## ✨ Planned Features
- User accounts and authentication  
- Create, edit and delete blog posts  
- Store all data in a database  
- Web interface using Symfony templates  
- Later deployment on a hosting platform  

## 🐳 Database via Docker
Docker is used only for the database, which allows:
- Easy setup  
- No local database installation  
- Consistent environment across systems  

Symfony connects to the Docker-based database.

## 🧠 Purpose of this project
This project was created to:
- Learn Symfony and MVC structure  
- Work with databases  
- Understand backend development  
- Prepare the project for later hosting  

It is still under development and not finished.

## 🛠 Technologies
- PHP (Symfony)  
- Docker (database only)  
- SQL database  
- HTML / Twig  

## 📌 Status
Early development – core structure is being built.
